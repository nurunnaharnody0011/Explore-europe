# Europe Explore

A responsive, modern travel website concept for European destinations.

## Features

- Homepage with hero section and premium travel styling
- Dark/light mode toggle with smooth animated transitions
- Search bar for filtering destinations
- Popular destinations section with floating glassmorphism cards
- Travel package pricing cards
- Animated statistics counters
- Europe travel image gallery with hover zoom effect
- Standalone login/signup flow using client-side storage
- Booking section with flight and hotel request prompts
- Responsive UI for desktop and mobile screens

## Files

- `index.html` - main website content and structure
- `style.css` - styling, responsive layout, animations, and theme rules
- `script.js` - interactive behavior, search, booking flow, auth state, and scroll reveal animations
- `auth.html` - login/signup page for user authentication flow
- `auth.js` - client-side auth and redirect logic

## Local Setup

1. Open the project folder in VS Code.
2. Run a local web server in the project folder.
   - If Python is installed, run:
     ```bash
     python -m http.server 8000
     ```
3. Open your browser and visit:
   ```
   http://localhost:8000
   ```

## Notes

- This is a static demo site and authentication is handled locally with browser storage.
- The site is intended for prototyping and visual presentation rather than production use.
